# Durable Lessons

Lessons that survived multiple runs and should influence future scans.
A lesson may be added only after the same pattern appears in at least 3 evaluated
paper candidates **or** one severe safety failure.

## Format

Each lesson entry should follow this structure:

```
### [LESSON-ID] Title (date range of supporting evidence)

- **Pattern**: What was observed repeatedly
- **Evidence**: Links to report dates, outcome rows, or source rows
- **Action**: How this lesson changes future scan behaviour
- **Confidence**: 0–1 based on sample size and consistency
- **Status**: active | retired
- **Provenance**: source_name | source_tier | source_ts
```
